# Line Following Robot
A Line Following Robot is an autonomous robot that is able to follow either black line on a white suface or white line on black suface.

## Components List
| Quantity      | Component           | 
| ------------- |:-------------------:|
| 2             | 360 Servo	Motor     | 
| 1             | Arduino Uno         | 
| 1             | Line Sensing Module | 
| 1             | Boe Bot             |
